# THE LAST COVENANT
## Synopsis Editorial (Format Pitch Commercial)

---

**ACCROCHE COMMERCIALE (4eme de couverture)**

Dans un monde ou les dieux se sont devores eux-memes, la mort n'est plus une fin.

Vous etes le Dernier Pactise. Immortel malgre vous. Chaque mort vous ramene. Chaque resurrection vous corrompt. Chaque choix nourrit le De du Destin — l'entite chaotique nee du suicide divin qui vous a offert ce cadeau empoisonne.

Descendez dans les Abysses infinies. Sauvez qui vous pouvez. Sacrifiez qui vous devez. Mais rappelez-vous : a chaque cage, a chaque dilemme, vous perdez un fragment de votre humanite.

Et quand vous ne serez plus humain... que deviendrez-vous ?

**Un RPG narratif ou vos choix ne sauvent personne. Ils reverent seulement qui vous etes vraiment.**

---

**PITCH SCENARISTE (Version Longue - 1 page)**

**CONCEPT HIGH-CONCEPT :**
"Dark Souls rencontre The Trolley Problem dans un univers ou les dieux morts vous regardent devenir l'un d'eux."

**PREMISE :**
Il y a 500 ans, sept dieux ont decide de se suicider collectivement pour echapper a une menace cosmique. Leur mort a brise le monde. Les morts ne restent plus morts. Les vivants se transforment en demons. Le chaos a remplace l'ordre.

Vous etes le dernier survivant d'un pacte maudit : en echange de resurrections infinies, vous explorez les Abysses — donjons generes par le De du Destin, incarnation du dieu Chaos. Votre mission officielle : rassembler les Fragments de Dieux pour... quoi exactement ? Ressusciter les dieux ? Devenir un dieu ? Detruire le concept meme de divinite ?

Le De ne vous le dit pas. Il ricane. Il lance ses des. Il vous regarde choisir.

**LA MECANIQUE NARRATIVE CENTRALE : LES CAGES**

A intervalles reguliers, vous trouvez deux prisonniers :
- Un **humain pur** (faible, innocent, symbole de l'Avant)
- Un **mi-demon** (fort, corrompu, symbole de l'Apres)

Vous ne pouvez sauver qu'un seul. L'autre meurt.

Chaque choix augmente votre **Corruption** (0-100%) :
- Sauver l'humain pur = +5% (vous vous accrochez a l'humanite, mais elle vous affaiblit)
- Sauver le mi-demon = +15% (vous acceptez l'evolution, mais perdez votre ame)
- Sauver les deux (item rare) = +30% Stress (vous refusez de choisir, mais le poids mental vous brise)

**A 100% de Corruption, vous ne jouez plus le meme jeu.**

**LES TROIS ACTES (Structure Narrative)**

**ACTE 1 : QUI ETAIS-JE ?**
Vous vous reveillez sans memoire. Le De vous guide (ou manipule ?). Premiers combats. Premieres cages. Premiere corruption. Vous rencontrez Kael (humain pur, cherche sa soeur) et Zhara (pretresse dechue, brule de l'interieur). Ils rejoignent votre village nomade. Boss : The Keeper of Bones — un gardien de tombes devenu roi des morts.

**Revelation :** Vous n'etes pas le premier Pactise. Vous etes le dernier. Tous les autres sont morts. Vraiment morts. Leurs ames dissoutes.

**ACTE 2 : QUI SUIS-JE ?**
La corruption monte. Vos mains tremblent. Vos pensees glissent vers la cruaute. Le village grandit, mais certains PNJ vous craignent maintenant. Nouveaux ennemis : Orc Berserker, Succube Corrompue, Chevalier Sans Tete. Nouvelles cages : des choix impossibles (sauver un enfant ou dix adultes ? trahir un ami ou perdre un artefact ?).

Boss : The Betrayer King — un roi qui a vendu son royaume au De pour le sauver. Ironie cosmique : il regne sur des fantomes.

**Revelation :** Les Fragments de Dieux existent. Mais les rassembler pourrait ne rien ressusciter. Pire : cela pourrait tout detruire.

**ACTE 3 : QUI VAIS-JE DEVENIR ?**
Corruption >75%. Vous n'etes plus vraiment humain. Le De vous parle comme un egal. Certains ennemis ne vous attaquent plus. D'autres vous venerent. Le village se divise : ceux qui vous craignent vs ceux qui vous adorent.

Dernier donjon. Derniers choix. Dernieres corruptions.

Boss Final : **LE DE LUI-MEME** — combat philosophique plus que physique. Il ne veut pas vous tuer. Il veut voir si vous **comprenez**.

**LES TROIS FINS (Non-Exclusives en Adaptation)**

**FIN A : LA RUPTURE DU PACTE** (Corruption <25%)
Vous brisez le De. Vous brisez le cycle. Le monde redevient mortel. Vous mourrez, vraiment. Paix. Mais un enfant ramasse un fragment de de brise... Le cycle recommence ?

**FIN B : L'ASCENSION DEMONIAQUE** (Corruption >75%)
Vous fusionnez avec le De. Vous devenez le Huitieme Dieu. Dieu du Chaos Embrasse. Vous remodellez le monde a votre image. Immortel. Puissant. Seul. Et vous comprenez enfin pourquoi les Sept Dieux se sont suicides : l'ennui eternel.

**FIN C : LA TROISIEME VOIE** (Corruption exactement 50%, item secret trouve)
Vous reconstituez la Balance d'Ael'mora (deesse Equilibre). Vous devenez le nouveau Gardien de l'Equilibre. Ni dieu, ni mortel. Juge eternel. Vous observez. Vous pesez. Vous ne controlez rien. Mais vous etes... necessaire.

**POURQUOI CE PROJET EST UNIQUE**

**1. PHILOSOPHIE ACCESSIBLE**
Pas de jargon. Les questions metaphysiques (libre arbitre, identite, sacrifice) emergent naturellement des choix de gameplay/narration. Le lecteur/joueur/spectateur **ressent** le dilemme avant de le **comprendre**.

**2. CORRUPTION COMME ARC NARRATIF**
La jauge 0-100% n'est pas qu'une mecanique. C'est une **transformation visible** :
- 0-25% : Mains propres, dialogues humains, espoir
- 50% : Mains qui tremblent, pensees sombres, doute
- 75% : Yeux assombris, rires involontaires, acceptation
- 100% : Quelque chose d'autre

**3. LE DE COMME ANTAGONISTE SEDUISANT**
Pas un mechant cliche. Un **trickster tragique**. Il ricane, mais il souffre. Il manipule, mais il cherche un egal. Il teste, mais il espere que vous reussissez. Complexite emotionnelle = profondeur narrative.

**4. ZERO HEROS BLANCS**
Pas de "gentils parfaits". Kael cherche sa soeur, mais ignore les autres. Zhara a accepte la corruption pour survivre. Le Pactise (vous) tue pour avancer. Tout le monde est gris. Gris fonce, parfois. Mais gris.

**5. REJOUABILITE NARRATIVE (pour adaptation transmedia)**
Trois fins radicalement differentes = trois saisons potentielles, trois arcs, trois messages. Pas "bonne fin vs mauvaise fin". Plutot "quelle verite preferes-tu ?"

Trilogie. Chaque tome = 1 acte. Epilogues multiples selon choix implicites du lecteur (style "tournez page X pour Fin A").

RPG narratif rogue-lite. Chaque run = une vie du Pactise. Corruption persistante entre runs.

8 episodes/saison. 3 saisons = 3 fins. Audience choisit via sondages quelle fin explorer en profondeur (interactivite meta).

Format episodique. Chaque cage = un chapitre. Art style sombre (Berserk meets Bloodborne).
